# Coffee  Analysis

Программа для анализа  медианной суммы трат на кофе по каждому студенту за весь период сессии. Читает данные из CSV файлов и вычисляет средние затраты на кофе для каждого студента.


### Основные возможности:
- Чтение данных из нескольких CSV файлов
- Автоматическое объединение данных по студентам
- Вычисление средних значений
- Сортировка результатов по убыванию
- Красивое форматирование вывода с помощью `tabulate`
- Обработка ошибок и некорректных данных
- Модульная структура для легкого расширения
